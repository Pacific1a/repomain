Тз Сайт Контрактные Двигатели Япония Vin Магнит
Техническое задание (ТЗ)
На разработку сайта по продаже контрактных двигателей и автозапчастей из Японии
1. Общая информация о проекте

Цель проекта: Разработка коТз Сайт Контрактные Двигатели Япония Vin Магнит
Техническое задание (ТЗ)
На разработку сайта по продаже контрактных двигателей и автозапчастей из Японии

1. Общая информация о проекте
Цель проекта: Разработка коммерческого сайта для продажи контрактных двигателей и автозапчастей из Япании с акцентом на быстрый подбор по VIN-номеру и мгновенную передачу VIN клиентом в мессенджеры (WhatsApp, Telegram, Max).
Ключевая бизнес-задача: Максимально упростить путь клиента: VIN → 1 клик → отправка VIN менеджеру → персональный подбор двигателя.
Сайт должен работать как VIN-магнит и инструмент генерации заявок.
Примеры сайтов для анализа и вдохновения:
Garant-Motor
Autodoc
Mysakura

2. Домен и протокол
Доменная зона: .рф
Пример: лучшезапчасти.рф, контрактмоторы.рф
Обязательное требование:
Использовать HTTPS (SSL-сертификат)
HTTP-доступ должен автоматически редиректиться на HTTPS

3. Целевая аудитория
Владельцы автомобилей (в основном японских марок)
Мужчины 25–55 лет
Клиенты, ищущие двигатель «под ключ» с подбором специалистом
Пользователи, которые не хотят разбираться в артикулах и каталогах

4. Ключевая функция — VIN-магнит (основное требование)
4.1. VIN-форма
На сайте (главная страница + каталог + отдельный блок) должна быть размещена форма:
Поле ввода VIN-номера (17 символов)
Подпись:
«Введите VIN-номер — подберём двигатель и запчасти»
Кнопка: «Подобрать двигатель» / «Отправить VIN»
4.2. Логика работы VIN-магнита
Сценарий пользователя:
Пользователь вводит VIN-номер
Нажимает кнопку «ОК / Подобрать»
Мгновенно появляется всплывающее окно (modal) с выбором мессенджера
Мессенджеры для выбора:
WhatsApp
Telegram
Max
4.3. Автоматическая подстановка VIN в сообщение
При нажатии на иконку мессенджера пользователь переходит в соответствующее приложение с уже сформированным текстом сообщения.
Текст сообщения (пример):
Здравствуйте! Мой VIN-номер: XXXXXXX Пожалуйста, подберите контрактный двигатель для моего автомобиля.
VIN должен автоматически подставляться из формы.
4.4. Технические требования к VIN-магниту
Работа без перезагрузки страницы (JavaScript)
Поддержка мобильных устройств
Корректная работа на iOS / Android / Desktop
Возможность изменения текста сообщения через админ-панель

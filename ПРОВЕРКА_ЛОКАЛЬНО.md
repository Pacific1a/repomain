# 🧪 Проверка работы локально

## ✅ Сервер запущен!

Ваш сервер работает на: **http://localhost:3000**

## 📋 Что проверить

### 1. Откройте в браузере

#### Главная страница бота:
```
http://localhost:3000/
```
**Должны увидеть:** Страницу с иконками игр (Roll, Crash, Mine и т.д.)

#### Сайт партнеров:
```
http://localhost:3000/partner/
```
**Должны увидеть:** Страницу "Duo Partners" с логином

#### Игра Roll:
```
http://localhost:3000/roll/
```
**Должна открыться:** Страница игры Roll

#### Main страница:
```
http://localhost:3000/main/
```

### 2. Откройте DevTools (F12)

В браузере нажмите **F12** → вкладка **Console**

**Если видите ошибки 404** для CSS/JS - скопируйте их сюда!

Пример:
```
❌ Failed to load resource: /pages_bot/main/style.css 404
❌ Failed to load resource: /config.js 404
```

### 3. Проверьте вкладку Network (F12)

1. Нажмите F12
2. Вкладка **Network**
3. Обновите страницу (F5)
4. Посмотрите какие файлы загружаются с ошибкой (красные)

## ❓ Если что-то не работает

### Проблема: Главная страница показывает сайт партнеров

**Решение:** Это нормально если вы открыли `/partner/`. Откройте именно `http://localhost:3000/` (без /partner/)

### Проблема: Стили не загружаются (404)

**Откройте файл:** `C:\Users\dev_fenomen\Desktop\bot\index.html`

Проверьте пути к CSS:
```html
<!-- ✅ Правильно -->
<link rel="stylesheet" href="pages_bot/main/style.css" />
<link rel="stylesheet" href="./pages_bot/main/style.css" />

<!-- ❌ Неправильно -->
<link rel="stylesheet" href="/pages_bot/main/style.css" />
<link rel="stylesheet" href="../pages_bot/main/style.css" />
```

### Проблема: Игры не открываются (404)

**Проверьте структуру:**
```
C:\Users\dev_fenomen\Desktop\bot\
├── index.html
├── roll\
│   └── index.html
├── main\
│   └── index.html
└── pages_bot\
    └── main\
        └── style.css
```

### Проблема: Партнерский сайт не открывается

Убедитесь что открываете: `http://localhost:3000/partner/` (с /partner/ в конце)

## 🚀 Когда всё работает локально

Если локально всё ОК, но на Render не работает:

1. **Закоммитьте изменения:**
```bash
git add .
git commit -m "Fix static files routing for bot and partner site"
git push origin main
```

2. **Render автоматически задеплоит** (если настроен Auto-Deploy)

3. **Проверьте логи на Render:**
   - Dashboard → Ваш сервис → Logs
   - Должно быть:
   ```
   ✅ Found bot index.html
   ✅ Found partner site index.html
   📁 Сайт партнеров раздается из: .../site
   ```

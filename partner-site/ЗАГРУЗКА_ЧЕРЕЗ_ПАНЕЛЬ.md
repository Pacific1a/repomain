# 🎛️ Загрузка сайта через панель управления

## 📋 Ваша панель: https://control.play2go.cloud/

Это **панель управления хостингом** - всё можно сделать через браузер! Без командной строки!

---

## 🚀 ШАГ 1: Войти в панель

1. Откройте: https://control.play2go.cloud/
2. Введите логин и пароль
3. Вы попадете в панель управления

---

## 📁 ШАГ 2: Найти файловый менеджер

В зависимости от типа панели:

### Если это ISPmanager (популярная панель):
1. В меню найдите **"Файловый менеджер"** или **"WWW-домены"**
2. Перейдите в раздел **"Сайты"** или **"Домены"**

### Если это cPanel:
1. Найдите **"Диспетчер файлов"** (File Manager)
2. Или **"Файлы"** в меню

### Если это другая панель:
- Ищите пункты: **"Файлы"**, **"File Manager"**, **"WWW"**, **"Сайты"**

---

## 📤 ШАГ 3: Загрузить файлы сайта

### Вариант A: Загрузить ZIP архив (БЫСТРЕЕ!)

1. В файловом менеджере найдите папку сайта:
   - Обычно это: `/var/www/html/` или `/home/username/public_html/`
   - Или `/home/username/www/`

2. Найдите кнопку **"Загрузить"** (Upload) или **"Закачать файл"**

3. Выберите файл:
   ```
   C:\Users\dev_fenomen\Desktop\bot\partner-site.zip
   ```

4. После загрузки найдите **"Распаковать"** (Extract/Unzip)
   - Кликните правой кнопкой на `partner-site.zip`
   - Выберите "Извлечь" или "Распаковать"

### Вариант B: Загрузить папку целиком

1. Выберите **"Загрузить папку"** или **"Множественная загрузка"**

2. Выберите все файлы из:
   ```
   C:\Users\dev_fenomen\Desktop\bot\deploy-partner-site\
   ```

3. Дождитесь окончания загрузки

---

## 🌐 ШАГ 4: Настроить домен/поддомен (если нужно)

### Если у вас есть домен:

1. В панели найдите **"Домены"** или **"WWW-домены"**
2. Нажмите **"Создать"** или **"Добавить домен"**
3. Укажите:
   - **Домен:** yoursite.com (или поддомен: partners.yoursite.com)
   - **Корневая папка:** путь где загрузили файлы

### Если тестируете по IP:

1. Узнайте IP сервера (обычно показан в панели)
2. Откройте в браузере: `http://ваш-ip-адрес/`

## 🔧 ШАГ 6: Настроить CORS на Render

Чтобы сайт мог обращаться к API на Render:

### На вашем компьютере:

1. Откройте: `C:\Users\dev_fenomen\Desktop\bot\server\server.js`

2. Найдите строку `app.use(cors())` (примерно строка 114)

3. Замените на:
```javascript
app.use(cors({
  origin: [
    'http://ваш-ip-или-домен',      // ← Замените!
    'https://ваш-ip-или-домен',
    'http://localhost:3000',
    'https://telegram-games-plkj.onrender.com'
  ],
  credentials: true,
  methods: ['GET', 'POST', 'PUT', 'DELETE', 'OPTIONS']
}));
```

4. Закоммитить:
```bash
git add server/server.js
git commit -m "Add CORS for test VPS"
git push origin main
```

5. Подождать 2-3 минуты пока Render задеплоит

## 🐛 Если что-то не работает

### Проблема: Не могу найти файловый менеджер

**Решение:**
- Сделайте скриншот главной страницы панели
- Покажите мне - я подскажу где искать

### Проблема: Файлы загрузились, но сайт не открывается

**Решение:**
1. Проверьте что файлы в правильной папке
2. Проверьте что `index.html` есть
3. Проверьте права на файлы (обычно 644 для файлов, 755 для папок)

### Проблема: Страница открывается, но без стилей

**Решение:**
1. Откройте DevTools (F12) → Network
2. Посмотрите какие файлы 404
3. Проверьте что все папки загрузились:
   - `js/`
   - `css/`
   - `duo-partner/`

### Проблема: CORS ошибки

**Решение:**
1. Обновили ли CORS на Render? (шаг 6)
2. Подождите 2-3 минуты после push
3. Обновите страницу Ctrl+Shift+R

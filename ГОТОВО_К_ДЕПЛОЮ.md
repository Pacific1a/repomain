## 📋 Деплой на Render - 3 простых шага

### Шаг 1: Закоммитьте изменения

```bash
cd C:\Users\dev_fenomen\Desktop\bot

git add .
git commit -m "Fix: Corrected file paths and routing for bot and partner site"
git push origin main
```

### Шаг 2: Настройте Render

Если у вас уже есть сервис на Render:
1. Перейдите на https://dashboard.render.com
2. Откройте ваш сервис
3. Render автоматически задеплоит после push (если включен Auto-Deploy)

Если нет сервиса:
1. Нажмите **"New"** → **"Web Service"**
2. Подключите GitHub репозиторий
3. Настройки:
   - **Environment:** Node
   - **Build Command:** `cd server && npm install`
   - **Start Command:** `node server/server.js`
4. Нажмите **"Create Web Service"**

### Шаг 3: Дождитесь деплоя

Render покажет логи деплоя. Должно быть:
```
✅ Found bot index.html
✅ Found partner site index.html
📁 Сайт партнеров раздается из: .../site
📁 Статические файлы бота раздаются из: ...
🚀 Сервер запущен на порту 10000
```



## 🐛 Если что-то не работает на Render

### Проблема: 404 на главной странице

**Решение:** Проверьте логи Render:
```
✅ Found bot index.html at: /opt/render/project/src/index.html
```

Если НЕ найден - проблема с `render.yaml` или структурой проекта.

### Проблема: Стили не загружаются

**Решение:** 
1. Откройте DevTools (F12) на Render сайте
2. Проверьте Network - какие файлы 404
3. Убедитесь что пути в HTML правильные (без `pages_bot/`)

### Проблема: Сайт партнеров 404

**Решение:** Откройте именно `/partner/` (со слешем в конце):
```
https://your-app.onrender.com/partner/
```

### Проблема: API не работает

**Решение:** Проверьте что в логах Render есть:
```
✅ Referral API endpoints loaded
```
